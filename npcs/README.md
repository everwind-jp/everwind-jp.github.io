# NPC概要

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)

## 概要

Everwindには友好的なNPC（ノンプレイヤーキャラクター）が存在し、交易や拠点の発展に関わります。

---

## NPCの種類

### 入植者（Settlers）

島やアイランドシップに住む一般的なNPCです。

情報収集中

### 商人（Merchants）

アイテムの売買を行うNPCです。各バイオームの島に存在する場合があります。

#### 商人の機能

- **購入**: 商人からアイテムを購入できる
- **売却**: 不要なアイテムを売却してゴールドを得る（情報収集中）
- **在庫**: 商人の在庫は時間経過で更新される（情報収集中）

#### The First Ascent（2026-05-28 配信）で追加された商人

- **Pirate Merchant（パイレーツ商人）**: 海賊船で出会える商人。海賊系のアイテムを取り扱う
- **Capybara Merchant（カピバラ商人）**: Capybara Island に稀に出現する珍しい商人。固有の珍品・装飾アイテムなどを扱う

> 📌 確認ソース: [Steam News - The First Ascent Update Has Arrived!](https://store.steampowered.com/news/app/2253100/view/673996212918026326) / [Massively Overpowered](https://massivelyop.com/2026/05/29/everwinds-first-major-update-the-first-ascent-adds-enemies-merchants-and-quality-of-life-improvements/)

### 衛兵（Guards）

島を警備するNPCです。モーティヴァーなどの敵と戦ってくれます。

情報収集中
