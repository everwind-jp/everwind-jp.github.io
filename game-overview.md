## Everwindとは

**Everwind**（エバーウィンド）は、空に浮かぶ島々の世界を舞台にした一人称視点のサバイバルRPGです。プレイヤーは自分だけの「アイランドシップ（Island Ship）」と呼ばれる飛行する島を拠点に、周囲のフローティングアイランド（浮遊島）を探索し、素材を集め、装備をクラフトし、強大な敵と戦います。

開発はポーランドの Enjoy Studio S.A.、パブリッシャーは Bohemia Interactive です。もともと「Skyverse」という名前で開発されていましたが、商標問題により「Everwind」に改名されました。Kickstarterで約53,000ドルの資金調達に成功し、2026年3月17日にSteam Early Accessとしてリリースされました。

## ゲームの柱

### 1. 飛行島シップ（Island Ship）

Everwindの最大の特徴は、プレイヤーの拠点が空飛ぶ島であることです。このアイランドシップは：

- **操縦可能**: 島ごと移動して、様々なフローティングアイランドに向かえる
- **カスタマイズ自在**: ブロック建築で自分好みの拠点を構築
- **アップグレード**: 速度・サイズ・高度をアップグレードで強化
- **クラフト拠点**: 各種クラフトステーションを設置して製作の中心地に

詳しくは → [飛行島シップ](ship/README.md)

### 2. クラス制RPG

3つのクラスから選択し、スキルツリーでキャラクターを成長させます：

- **ウォリアー（Warrior）**: 近接戦闘の専門家。剣と盾を構えて前線で戦う
- **アルカニスト（Arcanist）**: 魔法の使い手。杖を使った遠距離攻撃とサポート
- **エンジニア（Engineer）**: 機械と建築のスペシャリスト

詳しくは → [クラス一覧](classes/README.md)

### 3. クラフト＆建築

7種類のクラフトステーションで様々なアイテムを製作できます：

- **鍛冶（Smithing）**: 金属装備の製作
- **錬金（Alchemy）**: ポーション・マジックリキッドの調合
- **料理（Cooking）**: 食材から料理を作成
- **大工（Carpentry）**: 木製品・家具
- **ブロック（Block）**: 建築用ブロック
- **修理（Repair）**: 装備の修理
- **アップグレード（Upgrade）**: 装備の強化

レシピは探索中に発見したり、レシピスタンド（Recipe Stand）で習得します。

詳しくは → [クラフト概要](crafting/README.md)

### 4. 探索

6つの異なるバイオーム（地域）にまたがるフローティングアイランドを探索します：

- **森林（Forest）** - 序盤の基本バイオーム
- **ジャングル（Jungle）** - 熱帯の密林
- **沼地（Swamp）** - 危険な湿地帯
- **サバンナ（Savannah）** - 乾燥した草原
- **砂漠（Desert）** - 極度に乾燥した地帯
- **腐敗（Rotten）** - 汚染された高難度地帯

各島にはダンジョン、宝箱、敵の拠点などが存在し、プロシージャル生成により毎回異なるレイアウトになります。

詳しくは → [探索概要](exploration/README.md)

### 5. 戦闘

一人称視点での戦闘では、以下の手段を使います：

- **近接武器**: 剣・斧・棍棒など
- **盾**: 左手に装備してガード
- **魔法杖**: 火・電撃など属性ごとのスタッフ
- **弓矢**: 遠距離攻撃
- **錬金術**: マジックリキッドによるサポート

モーティヴァー（Mortivar）文明やスティーマー（Steamer）文明といった敵対勢力に加え、野生動物も脅威となります。

## ゲームの進行

Everwindはティア制の進行システムを採用しています。新しいバイオームを探索し、より高いティアの素材を集めて強力な装備をクラフトすることで、さらに危険な地域に挑めるようになります。

1. **序盤**: 森林バイオームで基本素材を集め、最初の装備を作成
2. **中盤**: シップをアップグレードし、新しいバイオームへ移動
3. **終盤**: 高ティア装備を揃え、強力なボスに挑戦

死亡するとデスオーブ（Death Orb）がドロップし、30分以内に回収しないとアイテムが消失するため注意が必要です。
