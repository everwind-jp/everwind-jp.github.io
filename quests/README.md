# クエスト概要

> 📌 情報源: [EverwindWiki - Quests](https://everwindwiki.com/quests/)

Everwindにはメインの探索・クラフト進行に加えて、村で受注できるサイドクエストが存在します。

---

## クエストの種類

| 種類 | 内容 |
|------|------|
| [サイドクエスト](side-quests.md) | 各村のボードから受注するコレクションクエスト。アイテムを集めてVillage StewardにXP報酬と交換 |

---
