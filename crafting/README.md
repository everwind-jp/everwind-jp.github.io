# クラフト概要

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)

Everwindのクラフトシステムは複数のクラフトステーションを中心に構成されています。各ステーション自体もクラフトで作成する必要があります。

## ステーション設置の優先順

序盤から順に設置すべきステーション：

1. **クラフトステーション** - 全ての出発点。最初に作る
2. **炉（Furnace）** - 鉱石を精錬してインゴットを作るため
3. **鍛冶ステーション** - 金属装備の製作に必須
4. **修理ステーション** - 装備の耐久値管理
5. **ブロックステーション** - シップの建築拡張
6. **料理ステーション** - 回復アイテムの安定供給
7. **大工ステーション** - 家具・収納の充実
8. **錬金ステーション** - 砂漠素材が必要なため、中盤以降

> ⚠️ 錬金ステーションの製作にはデザートパームプランク（砂漠バイオームの素材）とガラスが必要です。砂漠に到達するまでは設置できません。

## レシピの入手

レシピは以下の方法で入手できます：

1. **初期レシピ**: ゲーム開始時から一部のレシピが使用可能
2. **レシピスタンド（Recipe Stand）**: 島で見つかるレシピスタンドから習得
3. **ダンジョン探索**: ダンジョン内の宝箱からレシピを発見

## クラフトの基本フロー

```
素材の採集（島・ダンジョン）
    ↓
中間素材への加工（炉でインゴット化、プランク加工など）
    ↓
各ステーションでアイテム製作
    ↓
アップグレードステーションで性能強化
```

### 素材ティアの進行

```
木 / 石 / ロープ → 銅（Copper）→ ブロンズ（Bronze）→ 鉄（Iron）→ パイライト（Pyrite）
```

## ルーンクラフト

ルーンクラフトステーション（Rune Crafting Station）は通常のステーションとは異なり、レイス（Wraith）という敵を見つけて倒すことで入手・設置可能になります。ルーンはルーナリット（Runalit）の欠片とルーンフラグメント（Rune Fragments）を合成して作り、武器に特殊効果（攻撃速度UP・状態異常耐性など）を付与できます。

## 加工ステーション（Processing Station）

加工ステーションではアイテムを分解して基本素材に戻すことができます。不要な装備からインゴットを回収するなど、素材のリサイクルに使用します。
