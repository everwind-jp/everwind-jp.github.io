# 素材概要

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)

Everwindの素材は複数のカテゴリに分かれており、各バイオームで入手できます。素材はクラフト・建築・錬金・料理の基盤です。

---

|----------|-----------|----------|
| [鉱石・クリスタル](ores-crystals.md) | 金属装備・上位クラフト | 鍛冶・アップグレード |
| [木材](wood.md) | 建築・家具・基本ツール | 大工・ブロック建築 |

| [ルーン](runes.md) | 特殊効果の付与 | アップグレード |

## 素材の入手方法

### 採集

- **ピッケル**: 鉱石・岩を採掘
- **斧**: 木を伐採して木材を入手
- **素手 / ツールなし**: ハーブ・キノコ・フルーツは素手で採取可能

### ドロップ

- 敵を倒すと素材をドロップすることがある
- ボスは希少素材をドロップ

### ダンジョン

- ダンジョン内の宝箱から入手

### 農業

- 苗木（Saplings）と土壌（Soils）を使ってアイランドシップ上で植物を栽培

## ティア制の素材

素材にはティアがあり、上位バイオームほど高ティアの素材が入手できます。高ティアの素材からは高性能な装備を製作できます。

| ティア | 入手バイオーム | 素材例 |
|--------|---------------|--------|
| ティア1 | 森林（Forest） | 銅鉱石・基本木材 |
| ティア2 | 情報収集中 | 鉄鉱石・情報収集中 |
| ティア3 | 情報収集中 | 情報収集中 |
