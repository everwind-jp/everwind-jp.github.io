# 装備概要

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)

Everwindの装備は複数のカテゴリに分かれており、クラフトやダンジョン探索で入手できます。装備にはティア（Tier）があり、上位ティアほど性能が高くなります。

---

## 装備カテゴリ

| カテゴリ | 説明 | 詳細ページ |
|----------|------|-----------|
| [武器](weapons.md) | 剣・斧・棍棒・杖・弓など | 攻撃手段 |
| [防具](armor.md) | ヘルメット・チェストプレート・手袋など | 被ダメージ軽減 |
| [アティア](attires.md) | 布製の縫製防具。ソーイングマシンで製作 | 主にアルカニスト向け |
| [盾](shields.md) | 左手に装備してガード | 防御力強化 |
| [宝飾品](jewelry.md) | ネックレス・リングなど | 特殊効果付与 |
| [ツール](tools.md) | ピッケル・斧・ハンググライダーなど | 採集・探索 |

## 装備のティア

装備は使用する素材のティアに応じて性能が変わります：

| ティア | 素材例 | 入手バイオーム |
|--------|--------|---------------|
| ティア1 | 木材・銅 | 森林（Forest） |
| ティア2 | 鉄・情報収集中 | 情報収集中 |
| ティア3 | 情報収集中 | 情報収集中 |

## 装備の入手方法

1. **クラフト**: 鍛冶ステーション（Smithing Station）で素材から製作
2. **ドロップ**: 敵やボスからのドロップ
3. **宝箱**: ダンジョンや島の宝箱から入手
4. **商人**: 商人（Merchants）NPCから購入

## 装備の修理

装備には耐久値があり、使用するたびに減少します。修理ステーション（Repair Station）で素材を消費して修理できます。耐久値が0になると装備が壊れるため、定期的な修理を心がけましょう。

## 装備のアップグレード

アップグレードステーション（Upgrade Station）で装備の性能を強化できます。
