# 武器

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)

## 武器カテゴリ

Everwindには **10種類の武器カテゴリ** が存在します。装備ティアはウォリアー・アルカニストのスキルツリーで解放されます（Advanced Weapons: ティアIII、Master Weapons: ティアV）。

## 遠距離武器

### ボウ（Bows）

矢（Arrows）を消費して遠距離攻撃。4種類確認済み。

確認されている弓：

| 武器名 | 英語名 | ダメージ | 耐久値 | 備考 |
|--------|--------|---------|--------|------|
| アイアン補強木弓 | Iron-Reinforced Wooden Bow | 情報収集中 | 情報収集中 | 2026年3月29日のEverwindWiki更新で確認 |
| その他3種 | - | 情報収集中 | - | - |

ウォリアーのスキルで以下の強化が可能：
- **リサイクリング**: 矢を消費しない確率
- **スプリンティングエイム**: 走りながら弓を使える
- **マルチアロー**: 散弾のように複数矢を同時発射

### スペルブック（Spell Books）

魔法の範囲攻撃や特殊効果を使用できる武器。アルカニストのスキルで上位版が解放されます。

**バージョン 0.4.468 でバランス調整済み**: 各スペルブックの使用可能回数が増加し、一部のスペルブックは使用コストが削減されました。全体として以前より頻繁に使用できます。

## 武器の素材ティア進行

武器は使用する素材によってティアが決まります：

```
木 / 石 / ロープ（序盤）→ 銅（Copper）→ ブロンズ（Bronze）→ 鉄（Iron）→ パイライト（Pyrite）
```

### スティーマー武器

スティーマー（Steamer）文明の武器は、鍛冶ステーションのサブステーションである **ファブリケーター（Fabricator）** で製作します。

## 武器選びのポイント

| プレイスタイル | おすすめ武器 | 理由 |
|---------------|-------------|------|
| 攻防バランス | ソード + 盾 | ガードしながら攻撃可能 |
| 高火力近接 | ロングソード | 両手武器で最もダメージ効率が良い |
| 素早い立ち回り | ダガー | 攻撃速度が最速 |
| 遠距離物理 | ボウ | 安全な距離から攻撃 |
| 属性魔法 | ロッド / スペルブック | 元素ダメージとデバフ |
| 探索兼用 | ハチェット | 戦闘と伐採の両立 |

> ⚠️ Early Accessの情報に基づいています。武器の追加・バランス調整は今後のアップデートで行われる可能性があります。
