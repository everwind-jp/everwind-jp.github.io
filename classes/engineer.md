# エンジニア（Engineer）

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)

## 概要

エンジニア（Engineer）はユーティリティと探索に特化したスキルツリーです。ウォリアーやアルカニストが6ティアあるのに対し、エンジニアは現在 **1ティアのみ** 実装されています。探索・移動・索敵に便利なスキルが揃っており、効率的なプレイを支援します。

> 💡 クラスはロックされず、スキルポイントを自由に振り分け可能。エンジニアスキルをいくつか取りつつ、ウォリアーやアルカニストと組み合わせるハイブリッドビルドも有効です。

---

## スキルツリー（ティアI）

| スキル名 | 英語名 | 効果 |
|----------|--------|------|
| エリアスキャン | Area Scan | コンパスのスキャン機能でPOI（注目ポイント）をハイライト。射程300〜650m |
| ライフディテクション | Life Detection | 壁越しに生物をハイライト表示。射程最大650m |
| アイテムバリューチェック | Item Value Check | アイテムの価値を確認可能 |
| スプリントスタミナリダクション | Sprint Stamina Reduction | ダッシュ時のスタミナ消費を25〜100%軽減 |
| ベターツール | Better Tools | 上位のツールティアを解放 |
| サイレントムーブメント | Silent Movement | しゃがみ移動時に音を立てない |

## コンパスの使い方

エンジニアのエリアスキャン（Area Scan）スキルを取得すると、コンパスに新しい機能が追加されます：

- **左クリック**: コンパスのモードを切り替え
- **右クリック**: スキャンを実行し、周囲のPOI（注目ポイント）をハイライト

スキャン範囲はスキルレベルに応じて300m〜650mまで拡大します。ダンジョンの入口や宝箱の位置を事前に把握できるため、探索効率が大幅に向上します。

## エンジニアの価値

### 探索効率の向上

- **エリアスキャン**: 目的地を素早く特定
- **ライフディテクション**: 壁の向こう側の敵を事前に察知
- **スプリントスタミナリダクション**: 長距離移動が楽に（最大100%軽減で実質無限ダッシュ）

### 戦闘の間接サポート

- **サイレントムーブメント**: ステルスアプローチが可能に
- **ライフディテクション**: 敵の配置を事前に把握して有利なポジションを取れる

### リソース管理

- **ベターツール**: 上位のツールで採掘・伐採効率UP
- **アイテムバリューチェック**: 持ち帰るべきアイテムの判断に役立つ

## おすすめのハイブリッドビルド

エンジニアは1ティアしかないため、残りのスキルポイントをウォリアーかアルカニストに振るのが一般的です：

### エンジニア + ウォリアー

スプリントスタミナリダクション + エリアスキャンで機動力を確保しつつ、ウォリアーの戦闘スキルで火力を補う。

### エンジニア + アルカニスト

ライフディテクションで敵を事前に把握し、アルカニストの魔法で有利な位置から先制攻撃。

## 今後のアップデート

エンジニアのスキルツリーは今後のアップデートで大幅に拡張される予定です。現在の1ティアから、ウォリアー・アルカニストと同等の6ティアまで拡張される可能性があります。
