# 探索概要

> 📌 情報源: [Steam Store](https://store.steampowered.com/app/2253100/Everwind/)、[EverwindWiki](https://everwindwiki.com/)

## フローティングアイランドの世界

Everwindの世界は空に浮かぶ無数の島々（フローティングアイランド）で構成されています。プレイヤーはアイランドシップを操縦してこれらの島を訪れ、素材を採集し、ダンジョンを攻略し、敵と戦います。

## プロシージャル生成

フローティングアイランドはプロシージャル生成により、毎回異なるレイアウトで配置されます。これにより、探索のたびに新鮮な体験が得られます。

## 探索の基本フロー

1. **望遠鏡で偵察**: シップ上から望遠鏡（Spyglass）で目的の島を確認
2. **シップで移動**: アイランドシップを操縦して島に接近
3. **着岸・上陸**: 島に停泊してから上陸
4. **探索開始**: 素材採取・敵の討伐・ダンジョン攻略
5. **帰還**: 十分な素材を集めたらシップに戻る

## 完全破壊可能環境

Everwindの環境はボクセルベースで完全に破壊可能です。岩を壊して洞窟に入ったり、地面を掘って隠された宝を見つけたりできます。この自由度の高さが探索の醍醐味のひとつです。

## 探索時の注意点

- **デスオーブ**: 死亡するとデスオーブ（Death Orb）が30分で消失。貴重品はシップに保管
- **帰還手段**: ハンググライダー（Hang Glider）で素早くシップに戻れる
- **準備**: 食料・ポーション・予備ツールを持参
