# バイオーム

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)、[everwind.wiki](https://everwind.wiki/)

## 概要

Everwindの世界は **高度（Altitude）** によって3つのゾーンに分かれており、高度が上がるほど強力な敵と貴重な素材が出現します。アイランドシップの高度アップグレードで新しいゾーンに到達できます。

---

## 高度ゾーン

| ゾーン | 高度範囲 | バイオーム | シップコアレベル |
|--------|---------|-----------|-----------------|
| アルティチュード1 | 0〜400m | 草原・ジャングル | 初期 |
| アルティチュード2 | 400〜900m | 赤茶森林・沼地/墓地・腐敗島 | コアLv.2 |
| アルティチュード3 | 900〜1,800m | サバンナ・砂漠 | コアLv.3 |

---

## アルティチュード1（0〜400m）

### 草原（Starting Meadows）

| 項目 | 内容 |
|------|------|
| 難易度 | 低（最初のバイオーム） |
| 特徴 | 平坦な地形、散在する木々、川沿いの粘土 |
| 素材 | 木材、石、繊維、粘土、動物の毛皮、骨 |
| 鉱石 | 銅鉱石（地表の露頭） |
| 敵 | スケルトン（夜間パトロール、2〜3体グループ）、野生イノシシ、カエル、スパインジョー |
| 危険 | 夜間の低体温症（Hypothermia）、夜はスケルトンの攻撃性が上昇 |

**攻略ポイント**: 開始タワー（Starting Tower）には無料のかまど・炉・調理鍋が入っています。必ず全て回収してから出発しましょう。回収しないと序盤がかなり厳しくなります。

### ジャングル / グリムヴァー遺跡（Jungle / Grimver Ruins）

| 項目 | 内容 |
|------|------|
| 難易度 | 中低（アルティチュード2にも跨がる） |
| 特徴 | 密生した樹冠で地上が暗い、石のピラミッド、竹の村の廃墟 |
| 素材 | エキゾチックウッド、石炭（主要供給源）、竹 |
| 敵 | グリムヴァー（Grimver）文明の残存勢力 |

**重要**: グリムヴァーのピラミッド地下には **ルーンソケッティングテーブルの設計図（Rune Socketing Table Blueprint）** があります。

---

## アルティチュード2（400〜900m）

### 赤茶森林（Russet Forest）

| 項目 | 内容 |
|------|------|
| 難易度 | 中 |
| 素材 | 銅鉱石（豊富）、鉄鉱石、エキゾチック木材 |
| 敵 | エリートウルフ、スケルトンエクスキューショナー |

### 沼地 / 墓地（Swamp / Cemetery）

| 項目 | 内容 |
|------|------|
| 難易度 | 中高 |
| 特徴 | 半水没した島、濁った水、枯れ木、古い墓石 |
| 素材 | 骨など |
| 敵 | 全ゾーン中最多のスケルトン密度。墓地では5〜8体のスケルトンがパトロール。モーティヴァー沼地バリアント |

### 腐敗島（Rotten Island）

| 項目 | 内容 |
|------|------|
| 難易度 | **アルティチュード2で最も危険** |
| 特徴 | 廃棄された鉱山、崩壊した精錬所、露天掘りの発掘場 |
| 鉱石 | 鉄鉱石（最高密度）、石炭、レアメタルスクラップ |
| 敵 | スティーマー（Steamer）文明の残存勢力 |
| 危険 | **毒ガスゾーン**（約5HP/秒のダメージ）。毒耐性ポーションが必要 |

**重要**: スティーマーマシンパーツ（Steamer Machine Parts）がここでしか手に入りません。シップコアLv.2〜3のアップグレードに必須です。

---

## アルティチュード3（900〜1,800m）

### サバンナ（Savannah）

| 項目 | 内容 |
|------|------|
| 難易度 | 高 |
| 鉱石 | ティン（灰青色）、マナリス（紫色）、硫黄（淡い緑黄色） |

望遠鏡（Spyglass）で見ると、薄茶色の砂に対してコントラストで鉱石が見つけやすいです。

### 砂漠（Desert）

| 項目 | 内容 |
|------|------|
| 難易度 | **最高** |
| 特徴 | 極端な温度変化のある過酷な環境 |
| 鉱石 | ティン、マナリス、硫黄、金鉱石 |
| 素材 | レアクリスタル、古代ルーンフラグメント、上位設計図 |
| 敵 | アンデッドウォリアー、レイスバリアント、サンドクローラー |
| ボス | **巨大ゴーレムボス（Massive Golem Boss）** |
| 危険 | **二重温度危険**: 高高度の極寒（90秒で低体温症）**かつ**日中の熱中症 |

**重要**: 砂漠ダンジョンには **マジックリキッド（Magic Liquid）** があり、錬金ステーションの製作と錬金術に必須です。

---

## バイオーム攻略の推奨順

```
草原（Alt.1）→ ジャングル（Alt.1-2）→ 赤茶森林（Alt.2）→ 沼地/墓地（Alt.2）→ 腐敗島（Alt.2）→ サバンナ（Alt.3）→ 砂漠（Alt.3）
```

### 各ゾーンの移行タイミング

| 移行 | 条件 |
|------|------|
| Alt.1 → Alt.2 | シップコアLv.2にアップグレード。鉄インゴット＋古代メタルスクラップが必要 |
| Alt.2 → Alt.3 | シップコアLv.3にアップグレード。スティーマーマシンパーツ等が必要 |
