# オブジェクト概要

> 📌 情報源: [EverwindWiki - Objects](https://everwindwiki.com/objects/)（2026年4月13〜14日更新）

アイランドシップや島の建築・装飾に使用するオブジェクト類です。クラフトステーション（Crafting Station）・大工ステーション（Carpenter Station）・ブロックステーション（Block Station）などで製作します。
