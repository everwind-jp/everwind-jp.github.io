# 敵概要

> 📌 情報源: [EverwindWiki](https://everwindwiki.com/)

Everwindの敵は大きく3つのカテゴリに分かれます：野生動物、モーティヴァー文明、スティーマー文明です。

---

## 敵カテゴリ

| カテゴリ | 詳細ページ | 説明 |
|----------|-----------|------|
| [野生動物](wildlife.md) | 動物系の敵 | 受動的・中立・攻撃的の3種 |
| [モーティヴァー文明](mortivar.md) | 主要敵対勢力 | ボスを含む最大の脅威 |
| [スティーマー文明](steamer.md) | もうひとつの敵対勢力 | 機械系の敵 |
| [ボス攻略](bosses.md) | 強力なボス敵 | 高報酬・高難度 |

## 敵の行動パターン

### 受動的（Passive）

こちらから攻撃しない限り、襲ってこない動物です。素材ドロップ目的で狩ることが多いです。

### 中立（Neutral）

通常は無害ですが、攻撃すると反撃してきます。

### 攻撃的（Aggressive）

視界に入ると自動的に攻撃してくる敵です。モーティヴァーやスティーマーの兵士がこれに該当します。

## ドロップ

敵を倒すと素材やアイテムをドロップします。ボスは特に貴重なアイテムをドロップする可能性が高いです。
