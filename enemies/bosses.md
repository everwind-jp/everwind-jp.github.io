### レイスロード（Wraith Lord）

| 項目 | 内容 |
|------|------|
| 出現高度 | アルティチュード3（高度900ブロック以上） |
| 出現場所 | 暗緑色の沼植生と中央に暗い円形の墓地がある稀少な島。棺に近づくとボス戦開始 |
| HP | 約1,750 |
| 弱点 | **火属性**（最有効）、電撃属性 |
| ドロップ | 防具レシピ（ヘルム・チェスト・グローブ）、毒・氷耐性付き防具、ゴールド、ダスト、オーブ（防具セット製作に必要）、片手剣、特殊効果付きリング |

#### 攻撃パターン

| 攻撃名 | 内容 | 対処法 |
|--------|------|--------|
| 召喚攻撃 | 腕を天に掲げてレイスナイトの群れを召喚する（最も危険） | 速やかにナイトを処理してからボスに集中 |
| 弾丸攻撃 | 紫色の弾を2〜3連射する | 横にストレイフして回避 |
| 弾幕ブラスト | 腕を振り上げてから広範囲の弾幕を放射する | 大きく横移動で回避 |

#### 攻略アドバイス

- **火属性武器が最も有効**。アルカニストのファイアスタッフが特に効果的
- **レイスナイトを速やかに処理**しないと集団に圧倒される
- 棺（スポーン地点）から**離れすぎるとボスがリセット**するので注意
- **アイランドシップを近くに駐機**して遠距離から攻撃も可能
- 装備が十分でなければ、島の位置をメモして後回しにすること（複数の島で再出現する）
- レイスナイトはアイテムをドロップするため、十分な装備があれば撃破しながらファーム可能

> 💡 レイスロードの出現場所はSpyglass（望遠鏡）で島の植生・墓地のパターンを確認しながら飛行すると効率的に探せます。

## モーティヴァーの弱点と耐性

全てのモーティヴァー敵に共通する属性相性：

| 属性 | 相性 |
|------|------|
| 火（Fire） | **弱点 -10%** |
| 電撃（Electric） | **弱点 -5%** |
| 氷（Ice） | 耐性 +5〜10% |
| 毒（Poison） | 耐性 +5〜10% |

**結論**: モーティヴァー戦では **火属性が最も有効** です。アルカニストのファイアマスタリーが特に効果的。

## ボス戦の基本

### 準備

- 最高ティアの装備で挑む
- 回復ポーション・食事を大量に持参
- 武器の耐久値を確認（戦闘中に壊れると致命的）
- 予備武器を持っておく
- モーティヴァーボスには **火属性武器** が最も効果的

### 戦法

- **パリィを活用**: 成功するとスタミナ消費0で1.5秒のスタガー。最強の防御手段
- **ウォリアー**: 盾ガード + パリィでボスの攻撃を受け流しつつ反撃
- **アルカニスト**: 距離を保ちつつファイアスタッフで火属性ダメージ
- **Co-op**: ウォリアーがタンク、アルカニストがDPS。大幅に安全になる
